<h3>登录页面</h3>

登录页面的登录接口

（请求）2017.07.16
```
type: post
url: http://121.41.66.68/job/login
data:
说明：
1. 已登录时通过本接口发起请求，不需要传任何字段，会直接返回成功；
2. code:"200"成功，code:"0"失败

{
    // type: string
    // 用户名， length < 20
    // 未登录时必填
    username: "",

    // type: string
    // 密码， length < 20
    // 未登录时必填
    password: ""
}
```

（返回值）2017.07.15

```
// 登录成功时返回信息：
{
    code: "200",
    username: "当前登录的用户名"
}

// 漏填：
{
    code: "0",
    msg: "缺少用户名或密码"
}

// 用户名或密码错误
{
    code: "0",
    msg: "用户名或密码错误"
}

// 用户名或密码有敏感词
{
    code: "0",
    msg: "用户名或密码有敏感词"
}
```
